# CRM SaaS Platform

A modern, enterprise-style CRM SaaS frontend built with the MERN ecosystem using React, Vite, Zustand, Tailwind CSS, and advanced UI workflows.

This project focuses on delivering a real SaaS product experience instead of basic CRUD screens. The application includes a modern dashboard, Kanban-based sales pipeline, lead management workflows, dynamic analytics, reusable modal systems, drag-and-drop interactions, and scalable frontend architecture.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/crm-saas-platform.git
```

## Navigate Into Project

```bash
cd crm-saas-platform
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
